如何创建正确的变体？

第一个正确使用大小写的例子是当所有字母都是大写字母时。您需要使用Python string upper() method 将给定的line 转换为大写字母。这个方法返回一个新的字符串，你可以把它放在上面的括号中，而不用赋值给变量。 

return line in (line.upper(), ...)
如何进一步执行此策略？

正确使用大小写的第二个例子是当所有字母都是小写时。与上一步相同，但使用Python string lower() method 。将结果再次添加到括号内。

我们使用正则表达式创建三重模式，并从给定的line 的开头到结尾进行搜索。如果成立（line 与模式完全相等）--布尔值为True 。 

import re


def correct_capital(line: str) -> bool:
    
    my_regex = r"[A-Z]{1}[a-z]+\b|[a-z]+\b|[A-Z]+\b"
    result = re.match(my_regex, line)
    
    return bool(result)
