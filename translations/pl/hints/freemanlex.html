Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

ogólna idea polega na utworzeniu wszystkich wariantów podanego słowa z poprawnym użyciem przypadków i sprawdzeniu, czy rzeczywiste słowo jest równe jednemu z tych poprawnych wariantów. Jeśli line jest wśród poprawnych wariantów - proszę zwrócić True, w przeciwnym razie - False.

Ok, od czego powinienem zacząć?

Zgodnie z opisem mamy trzy poprawne warianty poprawnego użycia przypadku. Aby więc sprawdzić obecność danej line wśród innych wartości, można użyć operatora in bezpośrednio w instrukcji return. 

return line in (...)
Ok, jak utworzyć poprawne warianty?

Pierwszym przykładem poprawnego użycia przypadków jest sytuacja, w której wszystkie litery są wielkie. Proszę przekonwertować podane line na wielkie litery za pomocą Python string upper() method. Metoda ta zwraca nowy ciąg znaków, który można umieścić w nawiasie powyżej bez przypisywania go do zmiennej. 

return line in (line.upper(), ...)
Jak kontynuować tę strategię?

Drugim przykładem poprawnego użycia przypadków jest sytuacja, w której wszystkie litery są małe. Proszę zrobić to samo, co w poprzednim kroku, ale używając Python string lower() method. Proszę ponownie dodać wynik wewnątrz nawiasów. 

return line in (line.upper(), line.lower(), ...)
Jakiej metody powinienem użyć dla ostatniego przypadku?

Trzecim przykładem poprawnego użycia wielkości liter jest sytuacja, w której pierwsza litera jest wielka. Python ma również specjalną metodę do konwersji ciągu znaków na taki, w którym tylko wielka litera jest wielka - Python string capitalize() method. Proszę powtórzyć znany algorytm.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Proszę tylko opakować swój kod w wyrażenie funkcyjne. Zaleca się korzystanie z podpowiedzi typu.

Oto całe rozwiązanie:

def correct_capital(line: str) -> bool:
    
    return line in (line.upper(), line.lower(), line.capitalize())

Jasne, proszę łapać!

W tym rozwiązaniu użyto funkcji lambda (anonimowej). Ponadto, zamiast tworzyć nową poprawną wersję line i porównywać ją z oryginalnym słowem, sprawdzamy bezpośrednio, czy dane słowo jest w jednej z poprawnych form za pomocą określonych metod, które zwracają wartość logiczną. 

correct_capital = lambda a: a.isupper() or a.islower() or a.istitle()

Za pomocą wyrażeń regularnych tworzymy potrójny wzorzec i przeszukujemy go od początku podanego line do jego końca. Jeśli zostanie znaleziony ( line jest w pełni równy wzorcowi) - wartością logiczną jest True. 

import re


def correct_capital(line: str) -> bool:
    
    my_regex = r"[A-Z]{1}[a-z]+\b|[a-z]+\b|[A-Z]+\b"
    result = re.match(my_regex, line)
    
    return bool(result)
