Ok, what should I start from?

According to the description, we have three correct variants of correct case using. So in order to check the presence of given line among other values, you may use in operator directly in your return statement. 

return line in (...)
Ok, how to create correct variants?

The first example of the correct use of cases is when all letters are uppercase. You need to convert the given line into uppercase with Python string upper() method. This method returns a new string, which yom may put inside parenthesis above without assigning it to variable. 

return line in (line.upper(), ...)
How to proceed this strategy further?

The second example of the correct use of cases is when all letters are lowercase. Do the same like in the previous step, but use Python string lower() method. Add the result inside parenthesis again. 

return line in (line.upper(), line.lower(), ...)
What method should I use for the final case?

The third example of the correct use of cases is when the first letter is uppercase - capital letter. Python also has a specific method to convert a string into ones with capital letter uppercase only - Python string capitalize() method. Repeat the known algorithm.

So, how my final solution should look like?

You just need to wrap your code into function expression. Type hints using is highly recommended.

Here is the whole solution:

def correct_capital(line: str) -> bool:
    
    return line in (line.upper(), line.lower(), line.capitalize())

Using regular expressions we create triple pattern and search it from the beginning of the given line to its end . If it's founded (the line is fully equal to the pattern) - the boolean value is True. 

import re


def correct_capital(line: str) -> bool:
    
    my_regex = r"[A-Z]{1}[a-z]+\b|[a-z]+\b|[A-Z]+\b"
    result = re.match(my_regex, line)
    
    return bool(result)
